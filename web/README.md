# PEC Web Foundation

Prediction Engine Core Web Application Foundation

## Overview

This is the minimal web application foundation for Prediction Engine Core (PEC).

- React 19
- Vite
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Purpose

This foundation provides a Preview environment for reviewing UI changes during development.

No backend, prediction engine, or business logic is included in this foundation.
