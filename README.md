# Prediction Engine Core (PEC)

**Predict Better. Decide Better.**

Prediction Engine Core is a universal prediction engine built upon humanity's prediction models. It is a tool to reduce uncertainty about the future by integrating collective prediction knowledge.

## Mission

Prediction Engine Core is a tool to reduce uncertainty about the future.

## Vision

To become the world's prediction engine by integrating humanity's collective prediction knowledge.

## Core Principles

- **Models First**: PEC is defined by prediction models, not by prediction targets.
- **Simplicity**: Every design decision prioritizes clarity and simplicity.
- **Transparency**: All predictions and reasoning are explainable.
- **Curiosity**: We believe in continuous learning and evolution.
